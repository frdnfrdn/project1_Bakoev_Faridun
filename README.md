# Лабиринт сокровищ

Консольная текстовая adventure-игра на Python. Исследуйте лабиринт из 8 комнат, собирайте предметы, решайте загадки, избегайте ловушек и найдите сокровище!

## Описание

Игрок начинает во входе лабиринта и должен найти путь к сокровищнице. По пути его ждут:

- **8 уникальных комнат** с описаниями и предметами
- **Загадки** — решайте их, чтобы получить награды
- **Инвентарь** — собирайте предметы и используйте их
- **Ловушки** — случайные события, которые могут лишить предмета или жизни
- **Два пути к победе** — найти ключ или угадать код сундука

## Установка

```bash
make install
```

Или напрямую:

```bash
poetry install
```

## Запуск

```bash
make project
```

Или напрямую:

```bash
poetry run project
```

## Команды игры

| Команда          | Описание                                     |
|------------------|----------------------------------------------|
| `go <direction>` | Перейти в направлении (north/south/east/west) |
| `north/south/...`| Сокращение для `go <direction>`              |
| `look`           | Осмотреть текущую комнату                    |
| `take <item>`    | Поднять предмет                              |
| `use <item>`     | Использовать предмет из инвентаря            |
| `inventory`      | Показать инвентарь                           |
| `solve`          | Решить загадку в комнате                     |
| `help`           | Показать список команд                       |
| `quit`           | Выйти из игры                                |
